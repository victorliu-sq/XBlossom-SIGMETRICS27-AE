# 仁

境界：
求人（求不重要的人）=> 求己 => 求仁（求重要的人）。

关于仁，做大做强、无所畏惧的关键，在于做事时把对自己重要的人，放在自己的情绪、欲望和小利之前。

《论语》里所谓“事”，不是抽象的任务，而是一个人在人类社会中属于自己的人脉、位置和重要关系。所谓“利”，则是最后完全私人化、无法转化为关系和事业的蝇头小利，是个人一时的得失。

仁人，即君子、大人、境界高的人，能够约束个人欲望、情绪和小利，在人类社会中忠信于对自己重要的人，因此能立志取得巨大成就，做成巨大事业。相反，小人只专注于满足个人欲望、情绪和利益，不考虑那些对自己真正重要的人。

**志于道，据于德，依于仁，游于艺。**

依于仁：思考一件事，判断它重不重要，关键要看它背后的人。让自己愿意思考这件事，也要靠背后那个人的分量。

比如我讨厌做科研项目。每次想到庞大的项目、做出来的东西可能没人用、自己又不停换领域，就觉得抑郁。但是我想让张老师满意，想让张老师夸我会办事、做得好。张老师是一个大人，和张老师的联系让我变大。所以我去想张老师夸我的样子，就能克服对项目的不满和恐惧。

我讨厌写文章的费力，但每次想到我把事情搞明白，能够让父母放心，我就能克服敲文章的疲惫。

**仁者不忧，智者不惑，勇者不惧。**

仁是核心，智和勇是仁的表现。因为我们能看清并更看重那些对我们重要的人：大人、高人、贵人，是能让我们变大的人；友人，是对我们友好的仁；亲人，是对我们亲近的人。

因此，做事时不再忧愁和抱怨，因为注意力放在自己是否忠信于重要的人。也因此能够分清是非，知道什么事情可以做、什么事情不可以做，这就是智慧。还能战胜自己面对敌人和现实时的恐惧，这就是勇气。

特朗普被刺杀以后挥舞拳头，高喊“Fight, fight, fight”。他说自己想让支持者看到他的安全。这其实反映出他真心重视自己的支持者，所以能够克服个人被刺杀的恐惧。

木法沙对辛巴说，我更害怕失去你，所以我不害怕土狗。

张雪峰说，自己工作累得不行，但看到女儿熟睡的样子，就觉得再累也值。

朴正熙在陆英修被刺杀以后还能坚持对观众完成演讲，之后对朴槿惠说，如果你再出事，爸爸就真活不下去了。

这些人在困难和苦难面前表现出非凡的勇气、意志和刚毅，根本原因在于他们能看清对自己重要的人。所以他们不被自己的欲望和情绪困住，能够把自己的事业坚持下去。

这就是灭私奉公，或者说约私奉公的核心作用。公就是仁。在社会关系中，仁是一个人能否做大做强的关键。

游于艺：所谓艺，不只是艺术，而是一切事业中的心流状态。进入这种状态，就见招拆招，人挡杀人，佛挡杀佛，把任务拆解，一项一项解决，一不做二不休。

**不仁者不可久处约，不可以长处乐。仁者安仁，智者利仁。**

不仁者根本不知道在社会关系中谁对自己重要，也从来不在乎这些人。他既没有办法，也不愿意约束自己的情绪和欲望，更不愿意为对自己重要的人考虑。

欲壑难填。不断填充私人欲望的结果，必定是嫉妒、不满、心生怨恨。

智慧的人能够用仁让自己获得收益。我的理解是，把人际关系视作自己的财富，并且通过对重要的人忠信，让自己去做重要的事。当这个状态稳定下来，人也就安稳了。

每件事情背后都有人。这个人的大小，决定了一件事的大小和价值。

思考事情背后的人。如果背后是大人，那么让这个大人满意，让他夸我们会办事，就可以帮助我们约束欲望和情绪，取消对不重要事情的思考，从而达到非礼勿听、非礼勿视、非礼勿言。

比如我思考小学生活，就会想起苏喜军。小人的出现，会让我立刻停止对这件事的思考。

我思考 2221 的考勤，就会想起一群占我便宜的学生、黑人、越南人。这些人让我立刻停止对这件无意义事情的思考。

而我的中高考，依靠的是我希望有一个更好的自己，去向喜欢的女孩表白。所以我能坚持下来。

我的研究项目，依靠的是我希望让张老师满意、高兴、欣赏我。所以我能坚持下来，去做自己不熟悉的领域。

我能照顾猫三个月，是因为我希望让范满意，所以我可以咬牙坚持下来。

每一件事情的重要性，都是我根据它背后那个人的重要性来判断的。而让我坚持做下来的，也是我对人的忠信。

**不念旧恶，怨是用希。**

不念旧恶，不是无原则地忘记伤害，而是不让旧怨继续占据自己的心智。怨恨最危险的地方，是它让一个人持续把注意力投向已经不值得的人。能把心重新放回重要的人和重要的事上，怨自然就少了。

**吾未见刚者。**

真正的刚，不是脾气硬，也不是嘴硬，而是能约束自己的欲望、情绪和怨气，不被小利、小人、小事拖走。能把自己钉在重要的人和重要的事业上，在面对非凡的苦难和压力面前不被打到，永远不起气馁，这才是刚。特朗普在被包括自己的副总统和女儿背叛以后能立刻恢复状态开始集会拉票，在他刺杀以后能自己站起来向支持者示意自己的安全，这种刚强来自于对自己支持者的责任感和忠诚。

**君子坦荡荡，小人常戚戚。**

刘强东说，他的很多员工给他写邮件，说这也没意思，那也没意思，活着也没意思。这是太看重自己的个人底层欲望。当欲望长期满足以后，人反而会出现这种空虚。

这也解释了为什么发达国家自杀率高。不管是发达国家还是发展中国家，高认知都是稀罕物。只是发展中国家的基本需求更难满足，所以很多人还没到这种空虚自杀的阶段。

**唯仁者能好人，能恶人。**

只有仁者，只有真正看重人际关系的人，才能判断哪些人对我好、值得交往，哪些人不值得交往。因为他不是根据一时情绪判断人，而是根据这个人在关系中的位置、作用和忠信来判断人。

**苟志于仁，无恶也。**

**放于利而行，多怨。**

一个人排除怨念的关键，在于能不能摈除低端的个人感受、情绪和欲望，把自己在人类社会中真正重要的人放在第一位。

把慈父、慈母、恩师、恩友、贵人、支持者放在心上，把他们的感受放在第一位，人就快乐了。因为人的心不再被私人小利困住，而是有了可以安放的对象。

**君子喻于义，小人喻于利。**

君子能够明白大义，明白自己身处人类社会之中，应该怎么做才能积累人际关系的财富。小人只在乎自己的情绪、欲望和金钱。

**以约失之者，鲜矣。**

**一朝之忿，忘其身，以及其亲，非惑与？**

如果能约束自己的欲望和情绪，在乎真正重要的事情，以及事情背后真正重要的人，那么因为冲动、嫉妒、怨恨、感受不好而做错事的情况，就会很少。

**主忠信。**

**敏于事而讷于言。**

君子对人忠诚：一心为对方，也就是为自己重要的人考虑，让对方满意；同时言而有信。

然后敏捷地把与这个人有关的事情做完，而不是通过说嘴来表现自己把对方放在心上。真正的忠信不靠表态，而靠事情完成。

道德绑架：我可以牺牲自己的利益，只要恩惠落在对我重要的人身上。

让座可以，但你来强要不行。因为你是一个无赖，我不能牺牲自己来给予一个无赖恩惠。

**君子怀德，小人怀土；君子怀刑，小人怀惠。**

刑：通型。

君子考虑的是德行，小人只顾眼前的安逸生活。君子做事，会考虑自己的榜样，比如朴正熙大统领的准则和处事方式，思考如何笼络人心、建立秩序。小人做事，只考虑自己能获得多少恩惠。

**不患无位，患所以立。不患莫己知，求为可知也。**

**不患人不己知，患其不能也。**

不担心自己没有实际的工作位置，而担心自己有没有足够的认知和社会关系，让自己真正自立。

不害怕别人不了解自己，应该追求让自己变成值得被了解的人：一个言而有信、与人忠信的人。

**夫达也者，质直而好义，察言而观色，虑以下人。在邦必达，在家必达。夫闻也者，色取仁而行违，居之不疑。在邦必闻，在家必闻。**

达者喜欢义气，看重重要的人和重要关系，重情分。他会仔细考虑对自己重要的人的话和神色，思考如何对重要的人谦恭，把自己的位置放低，不傲慢。这样的人，在国家通达，在家也通达。

重视名声的人，则说自己看重人，行动上却只看重自己的获利，并且对自己的行为毫不怀疑。这样的人，在国家有名，在家里也有名。

比如吉米·卡特的人权政治中的剥夺财产，奥巴马政府 DEI 中的种族歧视：口头上取仁，行动上却违仁。

**先事后得，非崇德与。**

先考虑把重要的人的事情做了，再考虑自己的得失，这就是崇德。

**无欲速，无见小利。欲速则不达，见小利则大事不成。**

如果贪重自己一时的得失，盯着能拿多少东西，而完全不考虑事情背后的人际关系，那么大事不成。

**居处恭，执事敬，与人忠。**

在家居住不傲慢，对父母家人谦恭。在外面对恩师恩人，敬重地做事。与重要的人相处，忠心不二，宽恕重要的人的不足，不抱怨。

**君子而不仁者有矣夫，未有小人而仁者。**

君子对待自己的敌人或者不重要的人而不仁，比如欺骗、忽视，甚至消灭，这种情况是有的。但没有小人会真正对自己的朋友仁德。因为小人没有稳定的忠信，只有自己的欲望和利益。

**事君，敬其事而后其食。**

对待对自己重要的人，先考虑把对方的事情做完，让对方知道自己把对方放在心上，然后再考虑自己能获得多少东西。


**子张问崇德、辨惑。子曰：“主忠信，徙义，崇德也。爱之欲其生，恶之欲其死。既欲其生，又欲其死，是惑也。‘诚不以富，亦祗以异。’”**

徙义：按人做事。

**子张问政。子曰：“居之无倦，行之以忠。”**

行之以忠则居之无倦。也不会心生怨恨。仁者不忧。

# 礼

礼是仁的具体方式，是连接自己和重要之人的桥梁。人对自己重要的人，要通过礼表现恭敬心。

**颜渊问仁。子曰：“克己复礼为仁。一日克己复礼，天下归仁焉。为仁由己，而由人乎哉？”颜渊曰：“请问其目。”子曰：“非礼勿视，非礼勿听，非礼勿言，非礼勿动。”颜渊曰：“回虽不敏，请事斯语矣。”**

克己复礼：就是上文所说的灭除私欲，把注意力放回对自己重要的人身上。

非礼勿视，非礼勿听，非礼勿言，非礼勿动：
面对对自己重要的人，不符合礼的事情不要做。这样才能表现恭敬心。

这是一种逆向思维。我们未必清楚什么才完全合乎礼，什么才真正表现恭敬；但我们很容易知道，什么事不该做，什么话不该说，什么眼神、语气、动作会让重要的人失望，甚至失去信任。

**仲弓问仁。子曰：“出门如见大宾，使民如承大祭。己所不欲，勿施于人。在邦无怨，在家无怨。”仲弓曰：“雍虽不敏，请事斯语矣。”**

出门如见大宾，使民如承大祭:通过重要的人和重要的事情让自己。

己所不欲，勿施于人：同上“非礼勿视，非礼勿听，非礼勿言，非礼勿动” 

在邦无怨，在家无怨：对自己重要的人不要苛责。通过做事和想重要的人来让自己做事。
